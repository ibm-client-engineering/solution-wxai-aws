## Architecture Decisions

- User managed Openshift in AWS
- Cloud Pak for Data with the following components
    - watsonx assistant
    - Watson Discovery
    - OpenPages
    - watsonx.ai
    - Watson Studio
    - Watson Machine Learning
    - Watson Knowledge Catalog 
- Foundational Models
    - mixtral
    - llama2
    - ibm-granite-chat-v2.1
